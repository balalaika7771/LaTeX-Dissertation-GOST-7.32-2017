# LaTeX шаблон для диссертации по ГОСТ 7.32-2017

Короткий рабочий шаблон: кладёшь свои файлы в `src/`, запускаешь `make`, получаешь `build/main.pdf`.

## Быстрый старт

```bash
# Один раз перед началом работы
./setup.sh

# Сборка документа
make          # Полная сборка с библиографией
make quick    # Быстрая проверка без библиографии
make view     # Открыть готовый PDF
make clean    # Удалить папку build/ с временными файлами
```

Готовый PDF будет в `build/main.pdf`.

## Структура проекта

Твои файлы:

```text
src/
  chapters/    # Главы диссертации (текст)
  images/      # Рисунки
  appendix/    # Приложения
```

- Любой `.tex` в `src/chapters/` автоматически попадает в документ как отдельная глава (в алфавитном порядке по имени файла).
- Любой `.tex` в `src/appendix/` автоматически попадает как приложение.

## Изображения

Файлы картинок кладём в `src/images/` (например, `src/images/my_image.png`).

Минимальный пример вставки рисунка в главе:

```latex
\begin{figure}[H]
\centering
\includegraphicspath[width=0.8\textwidth]{my_image.png}
\caption{Описание рисунка}
\label{fig:my_image}
\end{figure}
```

- Путь к файлу указывать без `src/images/` — он подставляется автоматически.
- Поддерживаются форматы `png`, `jpg`, `pdf`.

## Таблицы

Простейшая таблица:

```latex
\begin{table}[H]
\centering
\caption{Описание таблицы}
\begin{tabular}{|l|c|r|}
\hline
Параметр & Значение & Единицы \\
\hline
T & 25 & \textcelsius{} \\
P & 1{,}0 & бар \\
\hline
\end{tabular}
\label{tab:example}
\end{table}
```

## Формулы

```latex
Текст перед формулой.

\begin{equation}
E = mc^2
\label{eq:einstein}
\end{equation}

Текст после формулы со ссылкой на неё: см. формулу~\eqref{eq:einstein}.
```

## Код

Для простых вставок кода можно использовать `verbatim` (подходит для кода с русскими комментариями):

```latex
\begin{verbatim}
def hello(name):
    # Пример функции
    print(f"Привет, {name}!")
\end{verbatim}
```

В приложениях уже настроен `lstlisting`, пример там можно скопировать и адаптировать под себя.

## Библиография и цитирование

- Файл с источниками: `bibliography/references.bib`
- В тексте: `\cite{ключ_источника}`
- Список источников выводится автоматически в конце (см. `main.tex`).

Пример:

```latex
Современные методы описаны в работе \cite{example_book}.
```

## Команды make

Основные команды:

```bash
make          # Полная сборка (LaTeX + biber + повторные прогоны)
make quick    # Быстрая проверка без библиографии
make view     # Открыть скомпилированный PDF
make clean    # Удалить папку build/ с временными файлами
make distclean # Полностью очистить всё, включая build/main.pdf
make errors   # Показать ошибки последней сборки
make warnings # Показать предупреждения последней сборки
make help     # Краткая справка по командам
```

## Что нужно установить

- LaTeX (TeX Live или MiKTeX)
- Biber (для библиографии)
- Make (для удобного запуска сборки)
